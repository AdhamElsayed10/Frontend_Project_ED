# مستقلين - Mustakleen

منصة متكاملة لخدمات الفريلانسر في مصر والوطن العربي.

## 🚀 المميزات

- ✅ **تأمين طبي** - تغطية صحية شاملة
- ✅ **تأمين مالي** - حماية الدخل ضد الظروف الطارئة
- ✅ **خصومات الكورسات** - تعلم مهارات جديدة بأقل التكاليف
- ✅ **خصومات المطاعم** - استمتع بحياتك الاجتماعية بأقل التكاليف
- ✅ **الترفيه** - خصومات على الجيمات والأنشطة

## 🛠️ التقنيات المستخدمة

- React 18
- Vite
- React Router DOM
- Tailwind CSS
- Framer Motion
- React Helmet Async
- Lucide React

## 🚀 تشغيل المشروع

```bash
# تثبيت التبعيات
npm install

# تشغيل في وضع التطوير
npm run dev

# بناء للإنتاج
npm run build

# معاينة البناء
npm run preview
```
